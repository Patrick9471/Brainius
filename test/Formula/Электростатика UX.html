Электростатика.Формулы по физике


1.1 Закон Кулона
$$F=k\frac{|q_{1}|\cdot |q_{2}|}{r^{2}}$$
F - Сила Кулона
q1 - Первый заряд
q2 - Второй заряд
k - Коэффициент пропорциональности
r - Расстояние между зарядами
1.2 Напряженность электрического поля
$$\vec{E}=\frac{F}{q_{0}}$$
Е – Напряженность электрического поля
q0 - электрический заряд.
1.3 Потенциал электрического поля
$$\varphi =\frac{W_{p}}{q}$$
q - заряд
1.4 Напряжение электрического поля
$$U=\varphi _{1}-\varphi _{2}$$
U - напряжение
1.5 Электроемкость
$$C=\frac{q}{\varphi}$$
C - электроемкость
q - заряд
1.6 Энергия заряженного конденсатора
$$W=\frac{CU^{2}}{2}=\frac{qU}{2}=\frac{q^2}{2C}$$
q –электрический заряд
U –напряжение между обкладками конденсатора
С –электроемкость конденсатора
